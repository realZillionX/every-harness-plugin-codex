# Changelog

## 0.1.0

- Added the Codex plugin manifest, public skills, hook bundle, installer, and shared mailbox runtime.
- Added built-in `fake`, `gemini-acp`, and `claude-cli` adapters.
- Added concrete ACP catalog coverage for OpenCode, OpenClaw, DeepSeek TUI, Kimi Code, Qoder CLI, Trae CLI, Qwen Code, GitHub Copilot CLI, Cursor Agent, iFlow CLI, Kiro CLI, Kilo Code CLI, and Factory Droid.
- Added Antigravity CLI as an explicit planned harness pending a verified headless or ACP contract.
- Added tests for runtime state, hooks, Gemini ACP behavior, and Claude CLI stream parsing.
