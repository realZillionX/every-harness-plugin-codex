# Changelog

## 0.1.0

- Added the Codex plugin manifest, single agent-facing Skill, `ehplugin` CLI, installer, and shared mailbox runtime.
- Added built-in `fake`, `claude-cli`, and native headless adapters.
- Added concrete ACP catalog coverage for OpenCode, OpenClaw, Qoder CLI, Trae CLI, Qwen Code, GitHub Copilot CLI, Cursor Agent, iFlow CLI, Kiro CLI, Kilo Code CLI, Factory Droid, and the community Pi ACP bridge.
- Added native headless coverage for Google Antigravity CLI, Kimi Code, and community CodeWhale / DeepSeek TUI.
- Added official Pi Coding Agent as an explicit planned harness pending a dedicated `pi --mode rpc` / `pi --mode json` adapter.
- Added tests for runtime state, Claude CLI stream parsing, generic ACP process spawning, harness catalog metadata, and native headless stream parsing.
