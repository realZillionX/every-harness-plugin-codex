# Changelog

## 0.1.0

- Added the Codex plugin manifest, single agent-facing Skill, `ehplugin` CLI, and shared mailbox runtime.
- Added built-in `fake`, `claude-code`, and native headless adapters.
- Added concrete ACP catalog coverage for OpenCode, OpenClaw, Qoder, TRAE, GitHub Copilot, Cursor, and Kiro.
- Added native headless coverage for Antigravity, Kimi Code, and community CodeWhale / DeepSeek TUI.
- Added tests for runtime state, Claude Code stream parsing, generic ACP process spawning, harness catalog metadata, and native headless stream parsing.
